# CoachFocus Web

## Run Locally
1. Install Node.js
2. Run `npm install`
3. Run `npm run dev`
4. Open http://localhost:3000

## Deploy to Vercel
1. Sign up at https://vercel.com
2. Create a new project and upload this folder
3. Deploy
4. Open the URL on your iPhone
5. Safari -> Share -> Add to Home Screen
